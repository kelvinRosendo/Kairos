# ⏱️ Kairos

> **Decida no momento certo.**

---

## 🚀 Sobre o Projeto

Kairos é um sistema de gestão financeira pessoal desenvolvido em Java, focado em ajudar usuários a tomar decisões financeiras mais inteligentes no momento certo.

A aplicação permite planejar, acompanhar e controlar o dinheiro de forma eficiente por meio de organização mensal, registro de transações e definição de orçamentos.

---

## 🧠 Ideia Central

> "O Kairos não mostra só o que você fez. Ele te ajuda a decidir o que fazer."

---

## 🎯 Funcionalidades (MVP)

- 📅 Planejamento financeiro mensal  
- 💸 Registro de transações (entradas e saídas)  
- 📊 Controle de gastos e saldo  
- 🎯 Definição de metas básicas  

---

## 🧱 Arquitetura

O projeto segue:

- Feature-Based Architecture  
- Clean Architecture (adaptada)
